## 发现问题？

如果在测试过程中发现问题：

1. **测试失败**
   - 查看详细错误信息
   - 检查数据库状态
   - 验证 API 响应
   - 使用 `npm run test:headed` 调试

2. **测试超时**
   - 增加超时时间
   - 检查网络连接
   - 关闭不必要的程序

3. **数据库错误**
   - 删除 database.db
   - 重新运行 `npm run init-db`
   - 确保没有其他进程占用数据库
